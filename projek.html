Aplikasi P5

Aplikasi QRIS Parkir

Sebuah aplikasi yang dilengkapi dengan fitur pemindaian barcode sebagai sistem parkir digital modern. Ketika barcode pada tiket atau kartu parkir dipindai, aplikasi secara otomatis menampilkan data identitas pengendara, seperti nomor SIM dan foto terdaftar. Teknologi ini memudahkan petugas dalam melakukan verifikasi, mempercepat proses masuk-keluar kendaraan, serta meningkatkan keamanan dengan memastikan setiap kendaraan terhubung dengan identitas yang valid.

QR Code Barcode Scanner Digital Identity Security
Jan 2025
Python App

Sistem Manajemen Kontak

Sebuah Sistem Manajemen Kontak yang dibangun menggunakan Python dengan fokus pada pengolahan data kontak secara cepat dan akurat. Aplikasi ini menyediakan fitur konversi data ke format VCF, pembagian kontak ke dalam banyak file, penomoran otomatis, serta opsi reset nomor antar file. Selain itu, sistem ini terhubung dengan Telegram bot sehingga pengguna dapat meng-upload file .txt, mengkonversinya, dan mengunduh hasilnya secara instan melalui antarmuka bot.

Python Telegram Bot VCF Automation
